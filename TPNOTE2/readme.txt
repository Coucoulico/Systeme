le programme copier qui permet de :
1-copier un fichier dans un autre 
2-copier un fichier dans un repertoire
3-copier un repertoire dans un repertoire

on peut l'installer seulement sur des systeme unix (linux ou mac)
pour l'installer il faut 
1-aller dans le repertoire 'TPNOTE2/installation'
2-executer le fichier setup.sh pour l'installation/unistall pour le desinstaller

dans le dossier source j'ai fournie 
1-mon code source en C dans le fichier main.c
2-les differents scripts bash pour l'installation la desinstallation et l'execution


-le fichier main.c contien un ensemble de fonction demandées(avec commentaires et precision) et une fonction test() pour tester le programme en generale.
-les deux repertoires dest et src c'est pour les different tests de la fonction test() dans main.c
!--pour executer la fonction test il faut avoir la commande tree à qui j'ai fait appel pour la visualisation des arborescence des resultat apres la copie de queque fichiers et repertoires,donc si vous ne l'avez pas elle ne s'executras pas



-la fonction main prend les deux parrametre 
        1-source:chemin vers le repertoire/fichier source
        2-destination:chemin vers le repertoire/fichier destination
